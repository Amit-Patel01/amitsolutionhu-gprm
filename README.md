# 🚀 Amit Solution Hub GPRM - GitHub Profile Maker

[![Next.js](https://img.shields.io/badge/Next.js-000000?style=for-the-badge&logo=next.js&logoColor=white)](https://nextjs.org/)

**Amit Solution Hub GPRM** is a premium, modern, and feature-rich GitHub Profile README generator. Built with a sleek **Glassmorphism UI**, it allows developers to create professional profiles with ease, featuring advanced customization not found in standard generators.

---

## ✨ Key Features

- **💎 Premium Glassmorphism UI**: A deep blue/purple aesthetic with vibrant gradients and glass-effect cards.
- **🖼️ 4-Point GIF Placement**: Choose exactly where your profile GIF appears:
  - Very Top (Above Title)
  - Top (Below Title)
  - Mid (After About Me)
  - Bottom (Above Footer)
- **✍️ Manual Customization**:
  - Type your own **Developer Quotes** manually.
  - Dedicated sections for **Vision & Mission**, **Expertise**, and **My Journey**.
- **⚡ Streamlined 5-Step Wizard**: A blazing fast flow that takes you from basic info to a ready-to-copy profile in seconds.
- **📱 Responsive Design**: Fully optimized for all devices, from desktop to mobile.
- **🛠️ Tech Stack Integration**: Support for hundreds of tech badges and icons.

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or later)
- npm or yarn

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Amit-Patel01/github-profile-readme-maker-main.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd github-profile-readme-maker-main
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Start the development server**:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
